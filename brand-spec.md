# PayLater Brand Spec

Extracted from user brief: "soft off-white canvases, deep midnight-black elements, and strategic coral-red accents."

## Tokens (OKLch)

- **--bg**: `oklch(93.6% 0.002 240)` (#EAEAEA - Canvas)
- **--surface**: `oklch(100% 0 0)` (#FFFFFF - Card Surface)
- **--fg**: `oklch(18.5% 0.01 260)` (#1C1C1E - Midnight Black)
- **--muted**: `oklch(55% 0.01 260)` (Muted Midnight)
- **--border**: `oklch(90% 0.005 240)` (Subtle Dividers)
- **--accent**: `oklch(62% 0.22 30)` (#EF4434 - Coral Red)

## Typography

- **Display**: Inter (Regular/Medium weights for metrics)
- **Body**: Inter
- **Mono**: JetBrains Mono (for numerics and meta)

## Layout Posture

- **Grid**: 16px outer edge margin.
- **Components**: Flat, non-shadow architecture.
- **Contrast**: Alternating white surfaces against grey canvas.
- **Hierarchy**: Large metrics (24px-32px) in Regular/Medium weights.
